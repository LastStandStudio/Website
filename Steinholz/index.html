My Favorite Programming Languages

These are my top 3 most used programming languages.

Java

Oh Java, this was my first official programming language, and I still find it very useful. Being both Cross-Platform and easy to use, I find many uses for it. I mostly use Java as an alternitive to a C-based language when I have a simple enough program to use. I also like Java over scripting languages because you can still get a little bit down and dirty with the code.

C++

While not being a joy to program in 100% of the time, C++ is definatly a strong language being portable and cross-platform. C++ can be a lot of fun or a lot of pain, there is not much in between. I mainly only use C++ when neccessary and tend to avoid it as much as possible.

Lua

Yes, Lua is a scripting language and not a programming language. I put Lua in my top 3 programming languages for it's ease of use and abilities. As of now I am using Lua in the Last Stand Studio Game Engine to do game specific code because Lua embeds into C++ with ease.

About me, Thomas Steinholz.

This is a small description of me

Hello, as of 2015 I am a high school student attending South River High. During the short period of time that I have been alive, I have spent most of my time programming and doing work on computers. I have built my own custom PC and my own private Linux distro with Arch Linux to dual boot with Windows 8.1 Pro. I have been programming for a few years now, I started out doing basic web projects and slowly promoted to programming with Java, modding minecraft and other fun stuff. Then eventually worked my way to C++ and learned some scripting languages along the way. I have done work as low as the Linux Kernel and as high level as game mods and websites. I feel like a well rounded, generic programmer, and it is my time to start to settle down in one specific field. That field is game development, I have always played video games from a very young age, and I have always programmed. It just makes sence, I have a ton of background knowlage not just how to make the game techinicaly but what is good in a game. I am still apart of this current generation which can make me better suited for finding the intrests of it versus the past generation. Everyday I learn something new, and I get just a litte bit better at what I do.
